# WebVR Babylonjs Webpack TypeScript Starter Project
Starter project using babylonjs, webpack, and typescript.

### Steps to Run Starter Project

`cd babylonjs-webpack-typescript-starter-project`

Install packages <br>
`npm install`

Build Project <br>
`npm run build`

Run the project <br>
`npm start`

Open in VS Code <br>
`code .`
